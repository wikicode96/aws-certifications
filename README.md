# aws-certifications
Apuntes de conceptos de AWS.

## Recursos recomendados

### Cursos y práctica
- [AWS Skill Builder](https://skillbuilder.aws/?utm_source=chatgpt.com)    
  Plataforma oficial de AWS con cursos, labs y simuladores.

- [AWS Cloud Quest](https://aws.amazon.com/es/training/digital/aws-cloud-quest/?utm_source=chatgpt.com)     
  Juego/laboratorio interactivo para practicar servicios AWS.

- [Tutorials Dojo](https://tutorialsdojo.com/?utm_source=chatgpt.com)  
  Muy bueno para mock exams y preguntas tipo examen real.

- [KodeKloud](https://kodekloud.com/?utm_source=chatgpt.com)  
  Labs prácticos y escenarios reales de DevOps y AWS.

## Ideas para practicar
- Crear una API con API Gateway + Lambda
- Desplegar un microservicio Spring Boot en ECS
- Practicar IAM Roles y Policies
- Crear una VPC simple con subredes públicas y privadas
- Usar RDS PostgreSQL
- Simular eventos con SQS y SNS
